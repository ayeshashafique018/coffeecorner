# ☕ CoffeeCorner  

### 🌿 A cozy and responsive coffee shop web app brewed with love — featuring menu browsing, search, cart system, and live bill receipts.  

---

## 🪄 Features  
- 🧋 **Dynamic Menu:** Displays coffee and beverage items fetched from JSON.  
- 🔍 **Search Bar:** Instantly filters items by name using JavaScript.  
- ➕ **Add to Cart:** Add one or multiple cups of any coffee.  
- 💰 **Generate Bill:** Automatically creates a receipt with total payment on checkout.  
- 🎨 **Aesthetic UI:** Soft brown, cream, and mocha tones inspired by cozy cafés.  
- 🧱 **Responsive Design:** Built using Bootstrap — perfectly adapts to all screens.  
- ⚙️ **DOM & JSON Integration:** Menu and cart dynamically managed with JS and JSON data.  
- 🧾 **Receipt Animation:** Bill section appears smoothly when payment is confirmed.  
- 🧭 **Navigation Bar & About Section:** Fully designed front page with info about the café.  

---

## 🖥️ Tech Stack  
| Technology | Purpose |

| **JavaScript (DOM)** | Dynamic behavior |
